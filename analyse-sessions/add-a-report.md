# Add a report

{% hint style="info" %}
This action is only available on a `Slides` document.
{% endhint %}

Go to an activity `Settings`, then click the `Ellipses` button and click `Add report to slides`.

This way, the activity overview and every session related to the activity will be added to the slides after the last slide.

{% hint style="warning" %}
The more sessions you have, the more time you have to wait.
{% endhint %}
